# Grupo 9-API para un Sistema de Alquiler de Vehículos
Lógica de negocio: Un cliente puede realizar una reserva de un vehículo para un rango de fechas. La API debe validar que el vehículo seleccionado esté disponible en esas fechas. Se debe gestionar el estado de los vehículos (Disponible, Alquilado, En Mantenimiento).
